# Applications and Software

## Operating System Components

An **Operating System (OS)** is system software that manages computer hardware and software resources. It allows users to interact with a device through graphical interfaces, command lines, drivers, and system services.

Modern operating systems are commonly designed for either **32-bit** or **64-bit** processors. A 64-bit OS can process larger amounts of data and memory more efficiently than a 32-bit OS.

### Main Types of Operating Systems

1. **Embedded Operating Systems**  
   Designed for dedicated devices such as:
   - Cars
   - Medical equipment
   - Smart appliances
   - Industrial systems

2. **Open Source Operating Systems**  
   Source code is publicly available and can be modified by developers. Examples include:
   - Linux
   - Android

3. **Commercial Operating Systems**  
   Proprietary operating systems that require a licence or purchase. Examples include:
   - Windows
   - macOS
   - ChromeOS

# File System Characteristics

A **file system** controls how data is stored, organised, and accessed on a storage device.

## Compression

Compression reduces the amount of storage space used by files by applying algorithms that store data more efficiently without losing information.

> Note: Files that are already compressed (such as `.mp3`, `.mp4`, or `.zip`) usually cannot be compressed much further.

## Encryption

Encryption protects files by converting data into a secure format that can only be accessed using the correct encryption key or password.

## File Types and Extensions

Different file types are identified using **file extensions**.

### Common File Categories

1. **Multimedia Files**
   - `.mp3`
   - `.mp4`
   - `.png`

2. **Document and Text Files**
   - `.txt`
   - `.docx`
   - `.pdf`
   - `.html`

3. **Data Files**
   - Spreadsheets
   - Databases
   - `.csv`

4. **Executable Files**  
   Programs and applications that can be run by the operating system.
   - `.exe` (Windows)
   - `.app` (macOS)
   - `.elf` (Linux)

5. **Archived Files**  
   Used to store multiple files in a compressed format.
   - `.zip`
   - `.rar`
   - `.dmg` (macOS disk image)

6. **Internet Files**
   Files used by web browsers and websites.
   - `.html`
   - Cache files
   - Cookies

# File Management

## Folders and Directories

File managers organise data into **folders** and **directories** to make files easier to store, locate, and manage.

## Permissions

Most operating systems use an **Access Control List (ACL)** to determine what actions a user or group can perform on a file or folder.

### Common Permission Levels

1. Full Control  
2. Modify  
3. Read and Execute  
4. List Folder Contents  
5. Read Only  
6. Write  

## Naming Restrictions

Operating systems place restrictions on file names.

### Common Rules

- File names are typically limited to **255 characters**
- Certain special characters cannot be used:

```text
? * / \ < > : |
```

# Disk Management

## New Technology File System (NTFS)

NTFS is Microsoft's file system used to organise and manage files on storage devices.

When a file is deleted, it leaves free space on the hard drive. New files may then be split and stored across different sections of the disk. Over time, this can lead to **fragmentation**, which may reduce performance.

### Defragmentation

Defragmentation is the process of reorganising fragmented files so they are stored together more efficiently, improving system performance.

# Interfaces

There are two main types of user interfaces that allow input into a computer system.

## 1. Command Line Interface (CLI)

A text-based interface between the user and the operating system.

Users enter commands using text. CLI systems are useful for:
- Older computers
- Fast navigation
- Advanced system management

### Advantages
- Low system resource usage
- Fast for experienced users
- Powerful system control

### Disadvantages
- Requires knowledge of commands
- Typing mistakes can cause errors
- Less user-friendly for beginners

---

## 2. Graphical User Interface (GUI)

A visual interface that allows users to interact with a computer using:
- Windows
- Icons
- Menus
- Pointers (mouse/touch)

This is the most common interface used on modern computers.

# Drivers

Drivers are software programs that tell the operating system how to communicate with hardware devices.

Examples include:

- Keyboards
- Audio devices

Drivers may occasionally become corrupted or require updates to improve compatibility and performance.

# Cache

Web browsers store **cache** files to improve performance and loading times.

When a website is visited, certain assets are downloaded and stored locally, such as:

1. Text
2. Images
3. Code files

This helps websites load faster on future visits.

However, cached data can sometimes become outdated or corrupted. Clearing the cache allows the browser to download updated files.
