# Infrastructure

- **Computing devices** = all have a motherboard, processer and storage but vary in how useres interact with input and output.

- **Desktop** = Personsal Computer (PC) that uses periphrials for input/output; e.i. Screens, keyboards, mouse and speakers. Pros: big and easy to fix/upgrade, large storage. Cons: Heavy, needs periphrials and requires power from a socket.
- **Laptops/notebooks** = Everything thing a desktop has but packed into a small case. Pros: Small and light so easy to move them, uses less electricty and has a battery. Cons: Difficult to fix, expensive and they are not as powerful as desktops.
- **Tablets** = Mini computer with a touchscreen, no keyboard and uses an OS. Hardware can not be upgraded and the OS can not be changed.
- **Hybrid laptops** = 2-in-1 touchscreen tablet with a keyboard. Very expensive.
- **E-Readers** = A type of laptop that uses e-ink so users can see the screen in the sun. Very good battery which can last for days.
- **Servers** = Powerful computers that store lots on information and can do multiple jobs for many users at one time. Has back up parts incase anything fails/breaks. Servers run continuosly and are not meant to be turned off.
- **Gaming Consoles** = Specail computers used for gaming. Normally they are connected via HDMI cables to TV's. Amazing graphics (graphic cards or GPU's) and uses ray tracking.
- **Argument Reality (AR)** = Puts computerized objects in the world around you.
- **Virtural Reality (VR)** = Takes user to a completely new world/environment.
- **Internet of Things (IoT)** = Devices that configure, report data or be managed remotely over IP addresses. These include: Home appliances, thermostats, security systems, home assistants, deadbolt doorlocks, video doorbells, vechicles, streaming devices, medical devices, exersise equitment, wearable devices.
