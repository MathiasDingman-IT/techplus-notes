#IT Concepts and Infrastructure
Input = information that is put into a computer (click, type, mouse)
Processing = what the computer does with the input (RAM is used as a volitile memory)
Storage = how the computer store the input either permanitly (hard drive/mass storage) or temporarly (RAM) 
Output = the information the computer gives back (screens, soound, printers, ie peripherals)

Computer Cylce = the process of how input is entered, processed, stored and how output is given to the user
