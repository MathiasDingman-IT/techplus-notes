# Data and Database Fundamentals

## Critical Data

- Highly important
- sensitive
- often protected

### Examples

- Finacial Records
- Password Databases
- Medical Data
- Payroll

Risks if lost or damaged

## Non-Critical Data

- less important
- easier to replace

### Examples

- Temporary Files
- Cache
- Drafts

Limited risk, inconveniance and loss of productivity

## Data-Driven Business Decisions

Companies collect data, analyse the data using algorithmns. Then the results/information to make more money. This process is known as **Big Data**. 

## Data Monetizing

- Data brokers take data, sort it and sell it.
- Anyone can make income from thier data by completeing things such as surveys for money, points or discounts.

## Data Base Concepts

- **Database** are a data storage system that organizes and store data into structured tables. many use SQL and security is a critical requirement. When crreating a database you must input data either manually or by importing it for other sources like spreadshhets, forms and other databases.
- **Quesries** is a request to get data from a database. Normally done in SQL.
- **Reports** are made when data from an Query is used and changed into something easy to read such as summaries, graphs and tables.
